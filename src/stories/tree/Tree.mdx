## Introudction

`TreePanel` 与 [TableCurd](?path=/docs/component-tablecrud--docs) 类似，是一个能够以树形结构都数据进行 CRUD相关操作，其内部通过在 [SEMIUI Tree](https://semi.design/zh-CN/navigation/tree) 基础上完成了**配置约定视图**。

- 基于 [SEMIUI Tree](https://semi.design/zh-CN/navigation/tree)
- 适用于树形侧
- 丰富的组件属性。
