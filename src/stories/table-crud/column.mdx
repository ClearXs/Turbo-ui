## common

`TableColumnProps` 继承于 [FormColumnProps（详细可以点击进入查看，在此就不列出相关属性）](?path=/docs/component-tform--column) 以及部分 [SEMIUI Table Column](https://semi.design/zh-CN/show/table)属性。

## TableInputColumnProps

<Canvas of={TableStories.Input} />

`[type=input]` 类型派生自 [FormInputColumnProps](?path=/docs/component-tform--column#forminputcolumnprops)

下面是其类型定义：

```tsx
type TableInputColumnProps<T extends Entity> = CardColumnProps<T> &
  FormInputColumnProps<T>;
```
