## Introudction

`TableCrud` 是一个能够快速搭建界面列表的组件，启发于 [formilyjs](https://formilyjs.org/)、[avue](https://avuejs.com/)、[amis](https://baidu.github.io/amis/) . 其强调的是**配置生成视图**。

目标是能够提供简洁、易拓展、高性能的 Table 组件。包含以下特性：

- 内嵌于 [SEMIUI Table](https://semi.design/zh-CN/show/table)。
- 灵活的 Table column 配置（详细看👉[column](?path=/docs/component-tablecrud--column)）。
- 丰富的组件属性。
- 内置 [TForm](?path=/docs/component-tform--docs)。
- 提供**分页**、**卡片**、**列表**、**树形**、**滚动**等多种模式视图。
