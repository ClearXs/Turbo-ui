## Introudction

`TForm(Template Form)` 是一个配置约定UI的组件, 通过高度配置自定义化, 以及引用 Formily 阿里出品的高性能表单解决方案，来能够为业务提供更为优质快速的表单构建解决方案，这些特性包括:

- 通用化表单字段配置(Column, 详细配置可以查看👉[column](?path=/story/component-tform--column)).
- 通过上下文表述表单行为, 包括**提交**，**验证**，**获取表单数据**... (更多详细看)[👉[context](?path=/story/component-tform--context)].
- 表单联动，引入 Formily 的 `schema-reaction` 机制来实现复杂的表单联动.
- 表单设计器，基于 Formily 的 Designable。
- 表单的UI使用于 [SemiUI](https://semi.design/) 一个现代化的设计系统。

> 什么是Formily❓
>
> [Formily](https://formilyjs.org/zh-CN) 是一个由阿里巴巴集团多BU 共建的面向中后台复杂场景的表单解决方案，它也是一个表单框架。

## Qucik Start

```tsx
import { Entity } from '@/api';
import { FormColumnProps, TForm } from '@/components/tform';

type Sample = Entity;

const sampleColumns: FormColumnProps<Sample>[] = [
  {
    field: 'id',
    type: 'input',
    label: 'id',
  },
];

<TForm<Sample> columns={sampleColumns} model="simply"></TForm>;
```
